Phase 1: Synthetic Data as a Thinking Tool

Description

In this phase, you’ll learn how to generate realistic, purpose-driven synthetic data using Python and spreadsheets. We’ll cover the tools and techniques to create data that mimics real-world scenarios, enabling you to think through problems strategically.

Video: Generating Synthetic Data with Python and Spreadsheets

This video walks you through the process of creating synthetic data using Python libraries like `faker` and `numpy`, as well as spreadsheet tools like Excel or Google Sheets, to simulate realistic business scenarios.

Explanation

Synthetic data must be purpose-driven—designed to reflect the specific problem you’re solving. For example, to analyze late deliveries, you might generate data with features like delivery distance, warehouse load, and time slot, ensuring the distributions (e.g., lognormal for distances) and relationships (e.g., evening slots are riskier) align with real-world patterns. In this exercise, you’ll use Python to generate a synthetic dataset and explore its properties, preparing you for the simulation challenges ahead.
