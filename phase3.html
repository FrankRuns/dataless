Phase 3: Capstone – Design for Judgment

Description

In this capstone project, you’ll tackle a real-world, high-stakes decision using synthetic data. You’ll create a dataset, run minimal analysis, and make a final recommendation backed by your simulation, applying all the skills you’ve learned in the course.

Video: Designing for Judgment with Synthetic Data

This video guides you through the capstone project, explaining how to design a synthetic dataset for a high-stakes decision, analyze it minimally, and make a confident recommendation.

Explanation

In real-world analytics, decisions often require judgment under uncertainty. In this capstone, you’ll face a high-stakes scenario: deciding whether to invest in a new delivery system to reduce late deliveries. You’ll generate a synthetic dataset with features like delivery times, costs, and customer satisfaction, run a minimal analysis (e.g., comparing costs vs. benefits), and make a recommendation backed by your simulation. The goal is to balance data-driven insights with strategic judgment.
